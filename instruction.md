# Телеграм-чат-бот «Обучалка английскому языку»
## EnglishCard
Курсовой проект по модулю "Базы данных для python-разработчиков"

### Цель проекта:
Создание телеграм-бота для изучения английского языка с использованием баз данных PostgreSQL.

### Описание проекта:
- Телеграм-бот помогает изучать английский язык по карточкам: пришлет слово на русском языке и даст 4 варианта ответа на английском, из которых необходимо выбрать правильный
- Есть функция добавления/удаления персональных слов для изучения
- Персональные слова сохраняются в базу данных

### Инструменты, необходимые для запуска бота:
- Python + IDE(Pycharm)
- PostgreSQL + PgAdmin
- Импортированные в IDE библиотеки: pyTelegramBotAPI и psycopg2-binary
- Зарегистрированный профиль в Telegram

### Прядок действий для запуска бота:
- В Telegram через @BotFather создать новый бот и получить персональный токен
- Создать базу данных на своем ПК и заполнить ее с помощью файла 'DataBase.sql'
- Заполнить файл 'config.py' вашими данными, необходимыми для подключения к базе данных и телеграм-боту.
- Запустить файл 'main.py' через IDE
